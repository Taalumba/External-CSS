Test page for external style sheet

In order to use this method of CSS modification to web pages, you must first create a separate style sheet containing all the styling and formating you want to apply to the said webpages. Make sure that the style sheet DOES NOT contain any HTML text, and that it is saved as a .css file

You must then go into the web pages that you want modified by the style sheet, and add a link to the style sheet in the head of the document.
